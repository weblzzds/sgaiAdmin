# 几个年轻人学点手艺

* 通过这个项目学习react
* 通过这个项目学习Ant
* 根据以往的项目，抽取了常用的东西出来，方便以后有类似的项目的时候能快速开发



## 模板

```
- Dashboard
  - Analytic
  - Monitor
  - Workspace
- Form
  - Basic Form
  - Step Form
  - Advanced From
- List
  - Standard Table
  - Standard List
  - Card List
  - Search List (Project/Applications/Article)
- Profile
  - Simple Profile
  - Advanced Profile
- Result
  - Success
  - Failed
- Exception
  - 403
  - 404
  - 500
- User
  - Login
  - Register
  - Register Result
```

## 开始

```bash
$ git clone https://github.com/gui-sukla/sgaiAdmin.git
$ cd sgaiAdmin
$ npm install
$ npm start         # visit http://localhost:8000
```




## 兼容

Modern browsers and IE11.
